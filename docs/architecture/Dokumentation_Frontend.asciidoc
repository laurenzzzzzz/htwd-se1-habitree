== 0. Öffnen im Container (Devcontainer)

Falls du mit Visual Studio Code arbeitest, kannst du das Projekt auch direkt im Container (Devcontainer) öffnen. Das bietet eine einheitliche Entwicklungsumgebung ohne lokale Einrichtung.

. Voraussetzungen:
	* Docker Desktop installiert und gestartet
	* Visual Studio Code mit der Erweiterung "Dev Containers" (bzw. "Devcontainer")

. Schritte:
 1. Öffne den Projektordner in Visual Studio Code.
 2. Klicke unten links auf "><" oder öffne die Befehlspalette (F1) und wähle "Reopen in Container" ("Im Container erneut öffnen").
 3. Warte, bis der Container gebaut und gestartet ist. Es wird bereits automatisch npx ci ausgeführt.
 4. Führe die gewohnten Befehle im Container-Terminal aus (z.B. npx expo start`).

== 2. Habitree - Expo App starten

Bitte beachten jeweils auf Handy und PC den eduVPN aktivieren, da der Backend Server im HTW Netzwerk ist.


=== 2.1 Lokal ohne Docker

[source,shell]
----
cd src/habitree
npm ci
npx expo install --fix <- Installiert und Updatet alle Packages
npx expo start --tunnel --go --clear
----

Scanne den angezeigten QR-Code mit der **Expo Go App** auf deinem Smartphone.
